Formatação de texto

Negrito/ Destaque

Para deixar o texto em negrito usamos a tag b (não semântica).

Agora para destacar um texto usamos a tag strong (que é semântica e faz o efeito de negrito.

Itálico/ Ênfase

Para deixar um texto em itálico usamos a tag i.

Agora a forma semântica expressa o sentido de ênfase, usando a tag em.

Texto marcado

Para marcar um texto como um marca texto usamos a tag mark

Texto grande e pequeno

A tag big é obsoleta, mas a tag small ainda pode ser usada.

Texto deletado e texto inserido

O texto deletado é aquele que pode ser lido, mas não deve ser considerado. Tag: del

O texto inserido é um texto que deve ser lido e deve prestar atenção.

Texto sobrescrito e subscrito

Sobrescrito: 22. Tag: sup.

Subscrito: O2. Tag: sub.

Código-fonte

Pega o texto é coloca em uma fonte mono-espaçada como um código normalmente é feito. Tag: code.

Pré formatado

        
Basicamente deixa o seu
            texto da forma que você colocou no code, todos os espaços, tudo. Tag: pre.

Citações

Citações Simples

Para usar uma citação simples devesse usar a tag q.

Citações Completa

Deixa a citação mais deslocada e usamos a tag blockquote.

Abreviações

Exemplo: HTML. Tag: abbr.
