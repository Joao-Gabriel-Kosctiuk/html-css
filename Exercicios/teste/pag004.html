Listas

Listas Ordenadas

São listas que expressam uma ordem. A tag para fazer uma é <ol> e além disso da para escolher com qual tipo de numeração vai ser utilizada com o atributo type. E para criar um item de lista é utilizada a tag <li>.

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não-Ordenadas

Listas que em si não tem ordem. Tag: <ul>.

Exemplo:

  • Item sem ordem
  • Item sem ordem
  • Item sem ordem

Listas Mistas

Listas ordenadas e listas não-ordenadas podem se misturadas com outras listas ordenadas e não-ordenadas.

Exemplo:

  1. Item 1
    • Item sem ordem
      1. Item 1
      2. Item 2
    • Item sem ordem
  2. Item 2
    • Item sem ordem
    • Item sem ordem

Listas de definições

São listas que o termo tem uma definição. Tag: <dl>, <dt>, <dd> usadas respectivamente.

Exemplo:

HTML
Linguagem de marcação de hipertextos, criada para criar sites na internet.
CSS
Linguagem para a estilização de páginas web.
Voltar
