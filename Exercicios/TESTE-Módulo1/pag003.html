Links/ Âncoras

Os hyperlinks permitem que você ligue um ponto a outro na WWW. Para criar um link é nescessário a tag <a>.

Links Externos

É um link que se redireciona para fora do seu site. É recomendado o uso do atributo target com o valor _blank que abre a página externa em outa janela.

Exemplo: Youtube

Link Interno

É um link que liga uma página do seu site para outra.

Um exemplo é o voltar, que volta para a página principal desse teste. *Olha lá embaixo e testa.

Link para download

Um link para download pode ser um aúdio, um arquivo word, um vídeo, uma imagem, tudo.

Exemplo: Baixe este aúdio
